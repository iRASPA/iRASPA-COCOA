Command-line

The iRASPA-app is accompanied by a command-line utility that is accesible from the terminal. A common scenario is to screen lots of structures using scripts (e.g. bash, python, perl). The executable is located at: 

  /Applications/iRASPA.app/Contents/Library/iRASPA-CLI/iRASPA-CLI.app/Contents/MacOS/iRASPA-CLI
The utility can read 'cif' and 'pdb'-files directly and allows fast computation of surface areas and helium void-fractions. Before use, the permissions for accessing files using the utility need to be setup in the preferences panel.

To allow convenient access to the utility create a symbolic link in /usr/local/bin/ 

  ln -s /Applications/iRASPA.app/Contents/Library/iRASPA-CLI/iRASPA-CLI.app/Contents/MacOS/iRASPA-CLI /usr/local/bin/iRASPA-CLI.app
  


Usage: iRASPA-CLI [options] [file ...]
  -s, --surfacearea
      Computes the surface area.
  -v, --voidfraction
      Computes the void fraction.
  -h, --help
      Prints a help message.
