Atoms-view

The atom-view shows the atom data of the selected frame (of the selected movie). The outline-view displays the atom data in tabular form. Atoms can be grouped together into hierarchical data. An outline view is a type of table which lets the user expand or collapse rows that contain hierarchical data.

The visibility per atom can be switched on and off using the visibility check-box. To set/unset all the checkboxes of the outlineview hold the option key when pressing any of the checkboxes. The contextual menu (right-mouseclick) shows symmetry operations, changing the visiblity to match the selection (i.e. all selected atoms are made visible, all others invisible), and scrolling the view to selected atoms, or top and bottom.
