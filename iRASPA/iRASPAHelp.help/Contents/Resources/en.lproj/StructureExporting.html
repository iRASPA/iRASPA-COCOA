Structure exporting

Structures can be exported from the Render-view and from the Atoms-view in CIF-, PDB, or XYZ-format. The export via the render-view exports all scenes and can result in multiple files, while the atom-view export exports only the atoms of the current movie-frame (always a single file).

Structure exporting in the Render-view

Exporting the structures from the Render-view can be achieved from the context-menu (right-click in the view). In general, there can be several scenes, each containing movies each containing one or more frames. Therefore, the exported files will be saved into a directory of which the name can be set in the save-panel. Each scene will be a different file, but depending on the format, the movies and frames might be combined in a single file (e.g. in the PDB-format).

Structure exporting in the Atom-view

Exporting the structures from the Atoms-view export the selected frame from the selected movie, and always is a single file. The filename can be set in the save-panel.
