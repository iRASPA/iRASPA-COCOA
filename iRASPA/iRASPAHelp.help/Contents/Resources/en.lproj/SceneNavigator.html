Scene navigator

The second tab of the navigators is the "scene-navigator". The scene-navigator is a list of "scenes", where each scene can contain "movies". Each movie contains "frames", where the frames for the selected movie can be viewed in the frame-navigator. Each movie has a "visibility" checkbox that toggle the visibility of the movie in the rendering. New movies can be added by drag and drop from another iRASPA document, or copied by copy and paste. Movies that are in the same scene-section are visible to eachother, i.e. ambient occlusion is computed per scene. Any properties set in the detail-panes are for all the frames of the selected movie.

Scenes can be a combination of movies of the different components of a system (for example: the cations, anions, and the water-solvent as three separate components), or they can be completely unrelated and place at any position relative to eachother. The latter can be used to make "collages". The origin of each movie (or frame) in world-space is set in "Cell"-detailpane and then "Cell/Boundingbox-properties".
