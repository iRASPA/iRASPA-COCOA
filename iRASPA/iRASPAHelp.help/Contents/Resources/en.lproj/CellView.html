Cell-view

In the cell-view, the properties of the unit cell and symmetry can be controlled. The properties can be viewed and set for multiple structures simultaneously in three ways:

  1. Using the project navigator.

    2. All frames of all movies of all scenes are used as the target for viewing and changing properties.

  2. Using the scene navigator for individual settings per movie.

    3. All frames of all selected movies of all scenes are used as the target for viewing and changing properties.

  3. Using the frame navigator for individual settings per frame.

    4. All selected frames of the selected movie are used as the target for viewing and changing properties.
